## Action
- Action 提交的是 mutation，而不是直接变更状态。
- Action 可以包含任意异步操作。

### 1.actions 定义
```
const store = new Vuex.Store({
  state: {
    count: 0
  },
  mutations: {
    increment (state) {
      state.count++
    }
  },
  actions: {
    increment (context) {
      context.commit('increment')
    }
  }
})
```
### 2. 分发 Action
- Action 通过 store.dispatch 方法触发(调用)
#### 2.1 载荷方式 和 对象形式分发
```
// 以载荷形式分发
store.dispatch('incrementAsync', {
  amount: 10
})

// 以对象形式分发
store.dispatch({
  type: 'incrementAsync',
  amount: 10
}))
```
